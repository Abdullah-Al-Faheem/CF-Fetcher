# Friends' Codeforces Submissions Viewer
A Firefox extension to quickly view friends’ accepted submissions on Codeforces.

## Features
- Fetch and view friends’ accepted submissions
- Lightweight and easy to use

## Installation
1. Open `about:debugging` in Firefox
2. Click **This Firefox** → **Load Temporary Add-on**
3. Select `manifest.json`

## Usage
1. Open Codeforces
2. Enter a problem page
3. click on the extension icon to view accepted solutions of friends

## Tech
- JavaScript, HTML/CSS
- Firefox WebExtensions API
